Add a Global Treasury Price Protection System to UStack.

OVERVIEW

UStack uses a single pooled treasury backed by Blink.

All user balances, vaults, and savings goals are tracked internally in the UStack database.

Price Protection is NOT configurable per user.

Protection applies to ALL users automatically.

The entire treasury will always be in one of two states:

* BTC Mode
* USD Mode

When Bitcoin drops beyond the protection threshold, the system automatically swaps the treasury from BTC to USD.

When Bitcoin recovers, the system automatically swaps the treasury from USD back to BTC.

All users participate equally.

DATABASE

Create a new table:

treasury_state

Fields:

* id
* current_mode (btc | usd)
* reference_price_usd
* protection_threshold_pct
* confirmation_minutes
* pending_action
* pending_started_at
* last_transition_at
* btc_balance_sats
* usd_balance_cents
* created_at
* updated_at

There should only be one active treasury_state record.
